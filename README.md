# 🤖 Cognitia AI Assistant

A full-stack conversational AI web application that allows users to interact with an AI-powered assistant.

## 📋 Project Overview

Cognitia AI Assistant accepts a user question, sends it to the Groq AI model, displays the response in a clean user interface, and stores the question and answer in MongoDB Atlas.

## 🛠️ Tech Stack

### Frontend
- React.js with Vite
- Axios for API calls
- Deployed on Vercel

### Backend
- Node.js with Express
- Groq SDK for AI integration
- Mongoose for MongoDB connection
- Deployed on Vercel

### Database
- MongoDB Atlas (Free tier)

### AI Model
- llama-3.1-8b-instant via Groq API (Free tier)
